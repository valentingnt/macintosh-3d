Materials

Materials are used to define how an object looks like. They are used to define the color, the texture, the transparency, the shininess, etc. of an object.

Three.js provides a lot of different materials. In this chapter, we will cover the most important ones.

Materials are defined in the three.module.js file. You can find them in the src/materials folder.

Let's start with the most basic material: the MeshBasicMaterial.
